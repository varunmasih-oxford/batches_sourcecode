# ✅ **SQL Topics – Category Wise (Complete List)**

## **1. DDL – Data Definition Language**

Used to define or modify the structure of the database.

* CREATE
* ALTER
* DROP
* TRUNCATE
* RENAME
* COMMENT
* Data Types
* Constraints

  * PRIMARY KEY
  * FOREIGN KEY
  * NOT NULL
  * UNIQUE
  * CHECK
  * DEFAULT
  * AUTO_INCREMENT / SERIAL

---

## **2. DML – Data Manipulation Language**

Used to manipulate data inside tables.

* SELECT
* INSERT
* UPDATE
* DELETE
* MERGE (in some DBs)
* RETURNING (PostgreSQL, Oracle)

---

## **3. DQL – Data Query Language**

Mainly used for reading/querying the database.

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* LIMIT / TOP / FETCH
* DISTINCT

---

## **4. DCL – Data Control Language**

Used to give or restrict database access.

* GRANT
* REVOKE

---

## **5. TCL – Transaction Control Language**

Used to manage transactions.

* COMMIT
* ROLLBACK
* SAVEPOINT
* SET TRANSACTION

---

## **6. Joins**

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* FULL OUTER JOIN
* CROSS JOIN
* SELF JOIN
* NATURAL JOIN

---

## **7. Subqueries**

* Single-row subquery
* Multi-row subquery
* Correlated subquery
* Nested subqueries
* EXISTS / NOT EXISTS

---

## **8. Set Operators**

* UNION
* UNION ALL
* INTERSECT
* EXCEPT / MINUS

---

## **9. Functions**

### **a. String Functions**

* CONCAT
* SUBSTRING
* LENGTH
* REPLACE
* UPPER / LOWER
* TRIM

### **b. Numeric Functions**

* ROUND
* CEIL
* FLOOR
* ABS
* POWER

### **c. Date & Time Functions**

* NOW()
* SYSDATE
* GETDATE
* DATEDIFF
* DATE_ADD / DATE_SUB
* EXTRACT(YEAR / MONTH / DAY)

### **d. Aggregate Functions**

* COUNT
* SUM
* AVG
* MAX
* MIN

### **e. Window Functions**

* ROW_NUMBER()
* RANK()
* DENSE_RANK()
* LAG() / LEAD()
* NTILE()
* OVER(PARTITION BY …)

---

## **10. Indexes**

* Creating indexes
* Unique index
* Composite index
* Full-text index
* Clustered index (SQL Server)
* Non-clustered index

---

## **11. Views**

* CREATE VIEW
* REPLACE VIEW
* Materialized Views (Oracle, PostgreSQL)

---

## **12. Stored Procedures**

* CREATE PROCEDURE
* Calling procedures
* IN / OUT parameters
* Error handling

---

## **13. Functions (User Defined Functions – UDF)**

* Scalar functions
* Table-valued functions

---

## **14. Triggers**

* BEFORE INSERT / UPDATE / DELETE
* AFTER INSERT / UPDATE / DELETE

## **15. Database Design Topics**

* Normalization (1NF, 2NF, 3NF, BCNF)
* ER Diagrams

# ✅ **Major SQL Database Systems (Languages)**

These are all SQL-based database systems (dialects) you should know:

## **1. MySQL**

* Very popular
* LAMP stack
* Used in web apps
* Free & open-source
* MySQL Workbench (GUI)

## **2. PostgreSQL**

* Most powerful open-source DB
* Advanced features (CTE, window functions, JSONB)
* Used in enterprise systems
* pgAdmin (GUI)

## **3. Microsoft SQL Server (T-SQL)**

* Used in enterprise & finance
* Version: Express, Developer, Standard, Enterprise
* SSMS (SQL Server Management Studio)

## **4. Oracle Database (PL/SQL)**

* Most powerful commercial database
* Used in banking & large corporations
* Oracle SQL Developer (GUI)

## **5. SQLite**

* Very lightweight
* Used in mobile apps
* No server needed

## **6. MariaDB**

* MySQL replacement
* Open-source
* Improved performance features

## **7. IBM Db2**

* Used for big enterprise systems
* Banking, insurance, mainframes

## **8. Snowflake SQL**

* Cloud data warehouse
* Uses ANSI SQL

## **9. Google BigQuery SQL**

* Cloud-based
* Uses Standard SQL

## **10. Amazon Redshift SQL**

* Data warehouse
* PostgreSQL-based SQL
